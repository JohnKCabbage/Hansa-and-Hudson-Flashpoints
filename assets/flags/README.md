# Flag asset placeholders

Replace placeholder assets by adding your own flag image files in this folder.

## File naming
Use lowercase code filenames with `.png`, for example:
- `ua.png`
- `ru.png`
- `xk.png`
- `un.png`

The app expects these code files:

`af, am, az, ba, bf, cd, cm, cn, co, cy, ec, eh, et, eu, ge, gy, ht, il, in, iq, ir, kp, kr, lb, ly, ma, md, ml, mm, mx, my, mz, ne, ng, pe, ph, pk, ps, rs, ru, sd, so, ss, sy, td, th, tw, ua, un, ve, vn, xk, ye`

## Fallback behavior
If a code image file is missing, the app will render `_placeholder.svg` automatically.
